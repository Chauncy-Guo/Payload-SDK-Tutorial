The functions supported by PSDK are divided into basic functions and advanced functions according to the difficulty of getting started.
#### Basic functions

* Log Management

* Information Management

* Payload Camera

* Camera Management

* Payload Gimbal

* Gimbal Management

* Power Management

* Flight Control

* Custom Widget

* HMS System

* Time Sync

* Data Transmission

#### Advanced functions

* Media File Metadata

* Video Stream

* Media Playback and Download

* Xport

* Liveview

* Local Upgrade

* Positioning

* SDK Interconnection

* Waypoint Mission

* Speaker
